# EcoAction - Environmental Action Website

A comprehensive website designed to help people learn how to help the environment through donating to trusted nonprofits and planting native species.

## Features

### 🌍 Environmental Education
- **Hero Section**: Compelling introduction to environmental action
- **Impact Cards**: Visual representation of how individual actions matter
- **Educational Resources**: Guides, videos, and tools for learning

### 💚 Donation Support
- **Trusted Organizations**: Curated list of reputable environmental nonprofits
- **Donation Impact Calculator**: Shows the real-world impact of donations
- **Direct Links**: Easy access to organization donation pages
- **Categories**: Organizations focused on reforestation, ocean conservation, wildlife, and climate action

### 🌱 Native Plant Guide
- **Regional Plant Database**: Native plants organized by US regions
- **Plant Finder Tool**: Filter by region, plant type, and sun requirements
- **Detailed Information**: Growing conditions, benefits, and care instructions
- **Benefits Education**: Why native plants matter for ecosystems

### 📱 Modern Design
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile
- **Smooth Animations**: Engaging user experience with scroll animations
- **Modern UI**: Clean, professional design with environmental theme
- **Accessibility**: Semantic HTML and keyboard navigation support

## Technology Stack

- **HTML5**: Semantic markup for structure and accessibility
- **CSS3**: Modern styling with Flexbox and Grid layouts
- **Vanilla JavaScript**: Interactive features without dependencies
- **Font Awesome**: Professional icons throughout the site
- **Google Fonts**: Modern typography with Inter font family

## Key Interactive Features

### Donation Impact Calculator
- Real-time calculation of environmental impact based on donation amount
- Different impact metrics for trees, ocean conservation, wildlife, and climate action
- Visual feedback showing the positive difference donations make

### Native Plant Finder
- Comprehensive database of native plants for all US regions
- Multi-criteria filtering (region, plant type, sun requirements)
- Detailed plant information including scientific names, growing conditions, and benefits
- Educational content about why native plants matter

### Smooth Navigation
- Fixed header with smooth scroll navigation
- Mobile-responsive hamburger menu
- Scroll-based header transparency effects
- Intersection Observer for scroll animations

## Getting Started

1. **Open the Website**: Simply open `index.html` in a web browser
2. **Explore Features**: Navigate through different sections using the menu
3. **Calculate Impact**: Use the donation calculator to see your potential impact
4. **Find Plants**: Use the plant finder to discover native species for your area
5. **Learn More**: Browse educational resources to expand your knowledge
